# 口语交际比赛系统 - 本地化版本

> 项目状态：规划中
> 创建日期：2026-04-26

---

## 项目目标

开发一款**完全本地化**的口语交际比赛应用：
- 所有数据存储在用户设备本地（IndexedDB）
- 无需网络连接即可使用
- 无需服务器，纯客户端应用
- 可通过微信内置浏览器或手机浏览器直接访问

---

## 核心技术

### 方案选择：Tauri + React

**推荐理由：**
- 桌面端 + 移动端 Web 一套代码
- React 做 UI，Tauri 打包成 App
- 数据完全存在本地，无服务器成本
- 可发布到 Windows/macOS/iOS/Android

### 备选方案：纯 Web + 内嵌 Bundle

如果只需要浏览器访问：
- React/Babel 内嵌到 HTML 中（不依赖 CDN）
- 所有功能同现有 index.html/admin.html
- 只是去掉服务器依赖，保留 CDN 依赖（但内嵌 bundle）

---

## 功能范围

### 后台管理（admin）
- 年段管理（创建/编辑/删除年级）
- 题目管理：
  - 认一认：上传图片作为题目
  - 辨一辨：上传大图 + 小图，标记正确答案
  - 说一说：上传图片作为题目
- 学生管理：添加/删除学生姓名
- 数据导入/导出（JSON 文件，方便备份迁移）

### 比赛模式（game）
- 随机点名选择学生
- 三环节完整流程：
  1. 认一认：展示图片 + 教师评分
  2. 辨一辨：找出相同类型 + 选择答案
  3. 说一说：展示图片 + 教师自由描述评分
- 成绩统计与历史记录

### 离线支持
- Service Worker 缓存所有资源
- 首次加载后完全离线可用
